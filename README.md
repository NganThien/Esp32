Mục đích:
Lập trình cho ESP32 gửi nhận dữ liệu qua MQTT
Sử dụng công cụ mô phỏng wokwi https://wokwi.com/
 
Hoạt động của chương trình:
a) Gửi dữ liệu thu thập từ sensor (nhiệt độ, độ ẩm) qua MQTT broker
b) Nhận dữ liệu qua MQTT broker, điều khiển LED on/off tương ứng 
Sử dụng MQTT broker có sẵn: broker.emqx.io
